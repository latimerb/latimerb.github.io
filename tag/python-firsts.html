This is a post about the receiver-operator characteristice (ROC) curve.

First, we generate two normal distributions, representing neuron responses to two different stimuli. Imagine one of them is recorded when a blue stimulus is applied and the other is when a green stimulus is applied. We wish to know if there is a clearly defined threshold for detection.

import numpy as np
import matplotlib.pyplot as plt

mu = 7 # Hz
sigma = 1 # Hz

# s_high has two distributions with high discriminability. That is, there is a big difference between the means.
s_high = np.zeros((1000,2))
s_high[:,0] = np.random.normal(mu, sigma, 1000)
s_high[:,1] = np.random.normal(mu-5, sigma, 1000)

s_med = np.zeros((1000,2)) #medium discriminability.
s_med[:,0] = np.random.normal(mu, sigma, 1000)
s_med[:,1] = np.random.normal(mu-2, sigma, 1000)

s_low = np.zeros((1000,2)) #low discriminability
s_low[:,0] = np.random.normal(mu, sigma, 1000)
s_low[:,1] = np.random.normal(mu, sigma, 1000)

%matplotlib inline

# Three subplots sharing both x/y axes
plt.figure
f, (ax1, ax2, ax3) = plt.subplots(3, sharex=True, sharey=True)
ax1.hist(s_high[:,0])
ax1.hist(s_high[:,1])
ax1.set_title('Three Firing Rate Distributions')
ax2.hist(s_med[:,0])
ax2.hist(s_med[:,1])
ax3.hist(s_low[:,0])
ax3.hist(s_low[:,1])
plt.xlabel('Firing Rate (Hz)')
# Fine-tune figure; make subplots close to each other and hide x ticks for
# all but bottom plot.
f.subplots_adjust(hspace=0)
plt.setp([a.get_xticklabels() for a in f.axes[:-1]], visible=False)

Clearly, for the "high" data set, somewhere between 4 and 6 will give us a nice threshold for detection, whereas the other datasets don't have such a clearly defined threshold. The ROC curve helps us quantify this intuition. To generate the ROC curve, we start by selecting a threshold, z, at a high value such as 10.

We define beta as the "hit rate". That is, the P(s > z|blue). This means the probability that the observed firing rate is higher than our threshold given that the stimulus was blue.

We define alpha as the "false alarm rate". That is, the P(s > z|green). That's the probability that the firing rate is greater than the threshold given that the stimulus was green.

With a large threshold, the probability that the firing rate will be larger than z is very low. Thus, alpha and beta start out as zero. As we decrease z, for the high discriminability case (top), beta quickly rises while alpha stays low. Let's see the implementation.

beta = np.zeros((14,3))
alpha = np.zeros((14,3))

q_b = np.zeros((1000,3))
q_a = np.zeros((1000,3))

for i in range(0,14):
    z = 14 - i
    q_b[:,0] = s_high[:,0]-z;
    q_b[:,1] = s_med[:,0]-z;
    q_b[:,2] = s_low[:,0]-z;
    beta[i,0]=np.size(np.where(q_b[:,0]>0))/100;
    beta[i,1]=np.size(np.where(q_b[:,1]>0))/100;
    beta[i,2]=np.size(np.where(q_b[:,2]>0))/100;
    
    q_a[:,0] = s_high[:,1]-z;
    q_a[:,1] = s_med[:,1]-z;
    q_a[:,2] = s_low[:,1]-z;
    alpha[i,0]=np.size(np.where(q_a[:,0]>0))/100;
    alpha[i,1]=np.size(np.where(q_a[:,1]>0))/100;
    alpha[i,2]=np.size(np.where(q_a[:,2]>0))/100;

plt.figure
plt.plot(alpha[:,0],beta[:,0],'r-*')
plt.plot(alpha[:,1],beta[:,1],'g-*')
plt.plot(alpha[:,2],beta[:,2],'b-*')
plt.xlim(-1,11)
plt.ylim(-1,11)
plt.xlabel('alpha (false alarm)')
plt.ylabel('beta (hit rate)')
Out[34]:

You can see that for the low discriminability case, the false alarm rate (alpha) and hit rate (beta) rise at the same rate, meaning there is no way to detect a difference. For the high discriminability case, the beta jumps up quickly while the false alarm rate stays low so we can make a more accurate prediction for this case.

From the figure, you can tell that the area under the curve is proportional to the level of discriminability. For this reason, you will sometimes see the term AUROC (Area Under the Receiver Operator Characteristic).
